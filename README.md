# Not Sky Force

## Running

To run on a local pc, open two seperate terminals.

Terminal one:

```
yarn install
yarn dev
```

Terminal two:

```
yarn start
```

Now open a web browser and the game will be live at `localhost:8000`!.

## Controls

We will obviously be using vim keybinds since they are superior.

`h` to move left
`l` to move right
`j` to move down
`k` to move up
`space` to shoot
